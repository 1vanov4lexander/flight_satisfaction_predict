# Предсказание удовлетворённости перелётом

## Цель проекта
Разработать модель логистической регрессии для прогнозирования удовлетворенности пассажиров авиаперелетов на основе демографических характеристик, оценок качества услуг и параметров полета.

## Использованные технологии
-Python
-Pandas
-NumPy
-Scikit-learn
-Seaborn
-Jupyter Notebook

## Краткое описание данных
Данные содержат информацию о 129,880 пассажирах авиакомпании, включая:
- Демографические характеристики (пол, возраст, тип клиента, цель поездки)
- Параметры полета (класс обслуживания, расстояние перелета, задержки)
- Оценки качества услуг (Wi-Fi, питание, комфорт сидений, развлечения, обслуживание)
- Социальные аспекты (удобство времени вылета, онлайн-бронирование)

## Ключевые выводы и результаты
- Проведена предобработка данных с целевым кодированием категориальных переменных
- Реализована модель логистической регрессии с градиентным спуском с нуля
- Достигнута точность прогнозирования 87.6% на тестовой выборке
- Определены наиболее значимые факторы удовлетворенности: удобство сидений, развлечения на борту и уровень обслуживания
- Модель может быть использована авиакомпаниями для улучшения качества услуг и выявления проблемных зон
